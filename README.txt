# -PS5-18-19-DiceForge-EquipeC
Dice Forge est un jeu de plateau de 2 à 4 joueurs.
La principale mécanique de jeu est que chaque face de chaque dé peut être modifiée.
Ce programme à pour but de représenter le jeu en version console.
Le jeu affichera les actions et les résultats de bots qui joueront entre eux.
